# m-index-archive

本项目是一个用于归档和发布多个独立 HTML 页面的仓库。  
所有页面均为静态文件，不依赖后端服务，可直接通过浏览器访问。

## 项目简介

本仓库主要存放各类独立编写的 HTML 页面，内容涵盖信息展示、工具界面、学习笔记等多种用途。  
每个页面均为自包含的 `.html` 文件，便于查看、分享和离线使用。

## 目录说明

- `index.html` — 总入口页面，提供归档页面的导航或列表
- `pages/` — 存放各独立 HTML 页面的子目录
- `assets/` — 页面所需的公共资源（如样式、脚本、图片等）

## 页面归档说明

- 每个页面以独立文件形式归档，文件名反映其内容或用途
- 页面中不包含外部跟踪代码、广告或自动跳转
- 归档页面可能不定期更新或新增，不保证所有页面长期可用
